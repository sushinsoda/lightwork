# Lightwork

From Python basics to LED installations, VJ projections and a visual product pitch for Spotify — 
built by an analytics engineer learning creative technology, one commit at a time.
Hopefully live events, installations and product launch in the future (fingers crossed).

## The Vision
Music and data as raw material. Light as output.
Building at the intersection of data engineering and creative technology.

## Project Arc
| Stage | What | Status |
|---|---|---|
| 1 | LED vase reacts to Python script | 🔨 In progress |
| 2 | Spotify data feeds the lights | ⏳ Coming soon |
| 3 | TouchDesigner visuals react to music | ⏳ Coming soon |
| 4 | Reactive visual skins by mood and genre — think Windows Media Player but actually beautiful | ⏳ Coming soon |
| 5 | Artist collaboration — artists build their own visual world for their music | ⏳ Coming soon |
| 6 | Users become their own VJ — a Spotify visual experience product pitch | ⏳ Coming soon |

## Stack
**Creative:** Python → Spotify API → TouchDesigner → Resolume
**Hardware:** Raspberry Pi Pico W → MicroPython → WS2812B LEDs
**Data & AI:** Spark → dbt → Snowflake → AWS → Airflow
